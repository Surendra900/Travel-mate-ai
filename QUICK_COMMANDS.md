# Quick Commands

## Start project

```bash
cd travelmate-ai
npm install
npm run dev
```

## Stop server
Press:

```text
Ctrl + C
```

## Build final version

```bash
npm run build
```

## Preview final version

```bash
npm run preview
```

## Git first setup

```bash
git init
git add .
git commit -m "initial TravelMate AI MVP"
```

## Create branch

```bash
git checkout -b safety-module
```

or

```bash
git checkout -b planner-module
```

## Save work

```bash
git add .
git commit -m "describe what you completed"
git push origin branch-name
```
