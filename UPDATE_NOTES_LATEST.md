# Latest TravelMate AI Feature Update

This version updates the planner around exact service visibility and safer booking positioning.

## Added

- Multi-transport boxes now show exact service names, train/flight/bus numbers, departure, arrival, fare, reliability, and status.
- Added an optimization permission step: the app asks "Can I suggest the best optimized route?" before highlighting the best route.
- Added detailed Rajahmundry/RJY → Ernakulam/Kochi → Karunagappalli/KPY demo connection data.
- Added all available local/offline demo services in the service board instead of showing only 1-2 options.
- Added direct in-app booking button as a Coming Soon flow.
- Added provider-entry packet copy button for IRCTC/airline/bus portals.
- Added clearer note that external-site auto-fill requires authorized provider integration or browser extension permission.

## Important demo rule

Do not claim live booking, live seat availability, live PNR, or guaranteed external-site auto-fill. This is an offline/local demo dataset and licensed-integration-ready booking preparation flow.
