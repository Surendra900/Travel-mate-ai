# Final refinement notes

- Low Network mode now has date, transport mode, passenger, class/cabin fields and a Book Ticket button.
- Low Network mode now shows service options for the selected transport instead of being train-only.
- First-time profile modal is scrollable and mobile-friendly.
- Lost Passport emergency flow/card was removed; SOS dropdown no longer includes Lost passport.
- Document Vault now supports custom display name/category for uploads, file naming based on that label, and rename after upload.
- Signal detection is less aggressive: good 4G/5G should not force Low Signal Mode unless the browser reports clearly bad conditions or the device is offline.
- Analyze Journey scoring now adds deterministic route-specific variation so different boarding/destination pairs produce visibly different demo scores.
- Voice Copilot includes TalkBack-style support with Read current page aloud, Read voice commands, and Stop reading buttons.
